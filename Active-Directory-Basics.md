# Active Directory Basics

## What is Active Directory?

Active Directory (AD) is Microsoft's centralized identity and access management service.

It helps organizations manage:

- Users
- Computers
- Groups
- Permissions
- Authentication

## Why is Active Directory Important?

Instead of creating accounts separately on every computer, administrators can manage everything from a central location.

## Example

User: Talha

Permissions:

- Email Access
- Company Applications
- File Shares

The account is managed centrally through Active Directory.

## SOC Analyst Perspective

When investigating incidents, analysts often review:

- User Logins
- Group Membership Changes
- New User Creation
- Privilege Escalation
- Domain Admin Activity
