# Cloud Security Basics

## What is Cloud Computing?

Cloud providers rent infrastructure and services over the internet.

Examples:

- AWS
- Azure
- Google Cloud Platform

## Cloud Service Models

### IaaS

Infrastructure as a Service

Example:
AWS EC2

### PaaS

Platform as a Service

Example:
Azure App Service

### SaaS

Software as a Service

Example:
Microsoft 365

## Shared Responsibility Model

Cloud Provider:

- Physical Security
- Hardware
- Infrastructure

Customer:

- Users
- Permissions
- Data
- MFA

## Important Security Services

- IAM
- MFA
- CloudTrail
- GuardDuty
- Security Center

## SOC Analyst Perspective

Investigate:

- New User Creation
- MFA Disabled
- Admin Role Assignment
- Suspicious Login
- Data Exposure
