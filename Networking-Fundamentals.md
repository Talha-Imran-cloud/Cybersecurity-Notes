# Networking Fundamentals

## What is Networking?

Networking allows devices to communicate and exchange data.

## OSI Model

1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application

## Important Protocols

- HTTP (80)
- HTTPS (443)
- DNS (53)
- SSH (22)
- RDP (3389)
- SMB (445)

## Public vs Private IP

Public IP:
Accessible from the Internet.

Private IP:
Used inside local networks.

## SOC Analyst Perspective

Investigate:

- Source IP
- Destination IP
- Protocol
- Port Number
- Network Traffic
