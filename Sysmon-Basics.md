# Sysmon Basics

## What is Sysmon?

Sysmon is a Windows system monitoring tool from Microsoft Sysinternals.

## Important Event IDs

Event ID 1
Process Creation

Event ID 3
Network Connection

Event ID 7
Image Loaded

Event ID 11
File Creation

## SOC Analyst Perspective

Monitor:

- PowerShell
- CMD
- Network Connections
- Malware Activity
