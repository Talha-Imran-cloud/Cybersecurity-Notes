# Incident Response Basics

## What is Incident Response?

The process of detecting, investigating, and responding to security incidents.

## Incident Response Lifecycle

1. Preparation

2. Detection

3. Analysis

4. Containment

5. Eradication

6. Recovery

7. Lessons Learned

## Example

Phishing Email
↓
PowerShell Execution
↓
Malware Installed
↓
User Created
↓
Data Theft

## SOC Analyst Tasks

- Review Logs
- Investigate Alerts
- Contain Threats
- Document Findings
